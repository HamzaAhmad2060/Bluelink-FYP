# Bluelink — Decentralized Bluetooth Chat App

> Selected for NUML Open House 2025 among top Final Year Projects

## About
Bluelink is a cross-platform mobile application that enables real-time 
peer-to-peer chat using Bluetooth — without any internet connectivity. 
Built for environments with limited or no network access.

## Features
- Real-time Bluetooth-based peer-to-peer messaging
- User authentication via Firebase
- Cross-platform support (Android & iOS)
- Clean and responsive UI/UX design
- Offline-first architecture — no internet required

## Tech Stack
| Technology | Purpose |
|---|---|
| Flutter | Cross-platform mobile development |
| Firebase | Authentication & real-time database |
| Dart | Programming language |
| Figma | UI/UX design & prototyping |
